# 🎖️ 최형우 전역 카운트다운 & 사진 아카이브

공군 입대(2026-06-22, 21개월 복무) 친구의 **전역까지 실시간 진행률**을 보여주고 놀리는 웹사이트.

## 기능
- ⏳ 전역까지 실시간 D-day (일/시간/분/초)
- 🔼 군생활 진행률 % — 소수점 7자리까지 실시간으로 차오름
- 😏 랜덤 놀림 멘트 (복사해서 카톡에 바로 전송)
- 📸 친구별 사진 아카이브 — 누구나 업로드, 닉네임 + 한줄드립 + 리액션(❤️😂🫡🔥)
- 👥 친구 추가 (육군/해군/공군/직접입력)

## 구조
- `index.html` — 단일 파일. 서버 코드 없음.
- 사진 아카이브는 **Supabase**(무료) 연동 시 동작 (`공유 모드 ☁️`). 설정 전엔 카운트다운만 도는 `로컬 모드 💾`.

## 설정 / 배포
- 사진 아카이브 켜기: [`SUPABASE_SETUP.md`](./SUPABASE_SETUP.md) 참고 (5분)
- 배포: `index.html`을 정적 호스팅(Netlify Drop / Vercel / GitHub Pages)에 올리면 끝.
